# Task Manager — Планировщик задач
## Описание проекта

Task Manager — это веб-приложение для управления задачами с поддержкой дедлайнов, приоритетов, проектов и тегов.
Приложение помогает пользователям эффективно организовывать работу с помощью канбан-доски, календаря и системы напоминаний.

Проект может быть расширен поддержкой совместной работы нескольких пользователей.

## Основной функционал
### Управление проектами
* Создание и редактирование проектов
* Удаление проектов
* Просмотр списка задач внутри проекта

### Управление задачами
* Создание задачи:
  * Название
  * Описание
  * Дедлайн
  * Приоритет (низкий / средний / высокий)
  * Статус (To Do / In Progress / Done)
  * Проект
  * Теги
* Редактирование и удаление задач
* Фильтрация задач по:
  * Статусу
  * Проекту
  * Тегам
  * Приоритету
* Сортировка по дедлайну и приоритету

### Теги
* Создание и удаление тегов
* Привязка нескольких тегов к задаче
* Фильтрация по тегам

### Календарь
* Отображение задач по датам дедлайна
* Дневной / недельный / месячный режим просмотра
* Быстрое создание задачи из календаря

### Канбан-доска
* Колонки по статусам:
  * To Do
  * In Progress
  * Done
* Drag-and-drop для изменения статуса
* Мгновенное обновление состояния

### Напоминания
* Уведомления перед дедлайном
* Email-уведомления (опционально)
* Настройка времени напоминания

### Роли и совместная работа (расширенная версия)
* Регистрация и авторизация пользователей
* Разграничение доступа:
  * Владелец проекта
  * Участник проекта
  * Совместное редактирование задач
* История изменений (опционально)
